# ⚾ FINANCE_PRO 2026

A high-performance, real-time personal finance dashboard built for the 2026 season. Designed for tracking family adventures, **Cubs Spring Training** in Arizona, and daily household wins.



### ⚡ The Tech Stack (Vibe Coded)
This project was built using **Vibe Coding**—prioritizing rapid iteration, intuitive flow, and a "neumorphism" aesthetic.

* **Frontend:** React + Vite for a blistering fast developer experience.
* **Styling:** Tailwind CSS with custom neumorphic glass-morphism.
* **Database:** Supabase (PostgreSQL) for real-time cloud synchronization.
* **Infrastructure:** GitHub + Vercel for "always-on" access from mobile.

### 🚀 Key Features
* **Live Cloud Sync:** Seamlessly transition from MacBook to iPhone. Add a transaction at Sloan Park, and see it on your desktop before you leave the stadium.
* **Family-First Budgeting:** Custom categories for the **boys** and household essentials.
* **Neumorphic UI:** A soft, modern interface that feels like the future of finance.
* **Real-time Heartbeat:** Powered by Supabase WebSockets for instant data replication.

### 🛠️ Local Setup
1. **Clone the repo** to your local machine.
2. **Run `npm install`** to grab the dependencies.
3. **Configure Environment Variables:** Create a `.env` file with your `VITE_SUPABASE_URL` and `VITE_SUPABASE_ANON_KEY`.
4. **Launch:** Run `npm run dev` and start vibing.
